HOW TO TEST NOW:

POSTMAN:

  2. URL: http://localhost:3000/server/catalyst_backend/api/auth/register
  3. Headers: Content-Type: application/json
  4. Body (raw JSON):
     {

       "Email": "testuser@railway.com",
       "Password": "TestPassword123!"
     }

  6. Check response for "details" field with error info

TROUBLESHOOTING:

If you see:
  • "CloudScale not initialized" → Wait 60 sec, restart catalyst serve
  • "Invalid request: JSON body required" → Check Body is set to raw JSON
  • "Invalid request: email and password required" → Check all fields present
  • "Email already registered" → Use different email
  • "Invalid email format" → Check email has @
  • "Password must be at least 6 characters" → Use longer password
