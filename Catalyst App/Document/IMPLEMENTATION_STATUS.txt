## How to Use It

### Create User via API

**Endpoint:** `POST /api/auth/register`

**Request:**
```json
{
  "Full_Name": "Test User Verification",
  "Email": "testuser@railway.com",
  "Password": "TestPassword123!",
  "Phone_Number": "9876543210",
  "Address": "Test Address, Test City"
}
```

**Response (201 Created):**
```json
{
  "success": true,
  "status": "created",
  "data": {
    "message": "User registered successfully",
    "user_id": "123456789",
    "email": "testuser@railway.com",
    "role": "User"
  }
}
```

---

### Sign In

**Endpoint:** `POST /api/auth/signin`

**Request:**
```json
{
  "Email": "testuser@railway.com",
  "Password": "TestPassword123!"
}
```

**Response (200 OK):**
```json
{
  "success": true,
  "status": "authenticated",
  "data": {
    "access_token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
    "refresh_token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
    "user": {
      "User_ID": "123456789",
      "Email": "testuser@railway.com",
      "Full_Name": "Test User Verification",
      "Role": "User"
    }
  }
}
```

---

### Via Frontend (Browser)

1. Open: `http://localhost:3000/app/auth`
2. Click "Register" tab
3. Fill in:
   - Full Name: Test User Verification
   - Email: testuser@railway.com
   - Password: TestPassword123!
   - Phone (optional): 9876543210
   - Address (optional): Test Address, Test City
4. Click Register
5. Form automatically switches to Sign In
6. Click Sign In
7. Redirects to dashboard on success
